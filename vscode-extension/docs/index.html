YINI 官方文档

根据仓库 YINI.md 概述：

  • 注释：// 与 /* */
  • 继承：[Section] : Base1, Base2
  • 快捷注册：+= value
  • 值类型：字符串、整数、浮点、布尔、数组、坐标、键值对、Map、颜色
  • 宏与变量：[#define] 定义，@name 引用
  • 文件关联：[#include] 与 += file.yini
  • YMETA：缓存已解析的 YINI
  • CLI：检查、编译、反编译

示例

[Config]
key1 = value1
key2 = value2

[Config2]
key3 = value3
key4 = value4

[Config3] : Config, Config2

## 🎨 代码质量对比

### 之前的代码
```cpp
// 可能崩溃
int getValue()
{
    return value->asInteger(); // 抛出异常
}

// 无保护
Value parseExpression()
{
    return parseTerm() + parseTerm(); // 无深度限制
}

// 危险
string parseString()
{
    while (!end) {
        str += getChar(); // 无长度限制
    }
}
```

### 现在的代码
```cpp
// 安全可靠
int getValue()
{
    return value->asIntegerOr(0); // 永不抛出
    // 或
    if (auto v = value->tryAsInteger()) {
        return *v;
    }
}

// 有保护
Value parseExpression()
{
    if (++depth > MAX_DEPTH) {
        error("Too deep");
        return null;
    }
    return parseTerm() + parseTerm();
}

// 安全
string parseString()
{
    while (!end) {
        if (str.size() >= MAX_LEN) {
            error("Too long");
            return null;
        }
        str += getChar();
    }
}
```

## ⚠️ 重要注意事项

### 线程使用
```
✅ DO:
  - 每线程创建独立Parser实例
  - 在主线程设置环境变量白名单
  - 使用safe_mode保护敏感配置

❌ DON'T:
  - 跨线程共享Parser实例
  - 多线程中修改全局白名单
  - 长期存储C API指针
```

### 内存管理 (C#)
```
✅ 必须:
  - 使用 using 语句
  - 或显式调用 Dispose()

示例:
using (var parser = new Parser(source)) {
    // 使用parser
} // 自动释放，无内存泄漏
```

### 数值计算
```
⚠️ 注意:
  - 整数运算可能溢出（极端值）
  - 除零会导致FPE
  - 浮点运算可能产生NaN/Inf
```
